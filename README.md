# Teaching Resources – Interactive Computational Materials

This repository contains **interactive computational teaching resources**, **primarily implemented as Jupyter notebooks**, but **not necessarily limited to them**.

The materials are used as **teaching support resources** for several undergraduate and graduate courses at the **Universidad Politécnica de Cartagena (UPCT)**.

These resources are designed for:
- interactive exploration
- simulation and modeling
- classroom demonstrations
- self-study and practice

👉 **No local installation is required**  
👉 Most resources can be executed directly in **Google Colab**

---

## 🚀 How to use these resources

Most resources in this repository are provided as **Jupyter notebooks** and can be opened directly in **Google Colab**.

### ▶️ Use without modifying the notebook (read-only mode)

If you only want to **run the notebook and interact with it** (execute cells, change widget values, explore results):

1. Click the **Open in Colab** button of the resource.
2. The notebook will open in **read-only mode**.
3. You can:
   - run all cells
   - execute individual cells
   - interact with sliders, buttons, and plots

✔️ **No Google account is required** for this use case.  
✔️ Changes are temporary and will be lost when you close the session.

---

### ✏️ Modify the notebook (save your own version)

If you want to **modify the code, change parameters permanently, or keep your work**:

1. Open the notebook using **Open in Colab**.
2. Create your own editable copy: File → Save a copy in Drive
3. Work on your copy stored in your Google Drive.

✔️ A Google account is required only for this step.

---

> ⚠️ **Important**  
> Always work on your own copy if you plan to modify the notebook.  
> The original version in this repository is read-only.
